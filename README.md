# PizzaPointWebApp
Pizza Point is a web app developed simple pizza restaurant or shop for selling pizzas .This app very web simple is very simple to use and user friendly .the app is PHP based app .The languages used in the development are as following HTML - CSS - Javascript - PHP - JSON - Jquery - bootstrap .

## Live Example here - https://pizzaappdemo.000webhostapp.com/
